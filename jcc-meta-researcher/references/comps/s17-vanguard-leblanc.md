# S17 重装妖姬

## Metadata

- Version: 17.2
- Season: S17 星神
- Tier: 高置信推荐
- Updated: 2026-04-30
- Sources: 金铲铲之战助手S17攻略合集；游侠手游S17重装妖姬乐芙兰；大地系统S17重装妖姬乐芙兰；17173转载官方17.2更新公告。
- Confidence: medium-high

## Core

- 核心羁绊: 4重装战士 + 3法官 + 3牧羊人 + 2暗星 + 2神谕 + 2旅人。
- 核心弈子: 乐芙兰、卡尔玛、努努、俄洛伊、小木灵、莫德凯撒、佐伊、蕾欧娜。
- 成型人口: 8成型；5阶段可上9补2星五费，或8级追三星四费。
- 成型阵容: 妖姬、卡尔玛、努努、俄洛伊、小木灵、铁男、佐伊、蕾欧娜。
- 可替换弈子: 过渡期佐伊/提莫带妖姬装备；后期按五费质量补控制或法系副C。

## Positioning

- 默认站位: 努努/蕾欧娜/俄洛伊前排，妖姬后排安全位持续输出，卡尔玛避开主控补伤害。
- 对刺客/切后排: 妖姬不要裸角，旁边放卡尔玛/佐伊吸引切入。
- 对AOE: 妖姬与卡尔玛分散，避免同时吃控制。
- 对单点爆发: 努努主T对位敌方主C，俄洛伊带日炎贴近人群。
- 站位注意事项: 核心是让妖姬持续叠输出，不要为了贪角落让她被秒。

## Items

- 主C装备: 妖姬羊刀、法爆、大天使。
- 主T装备: 努努狂徒、冰甲、冕卫。
- 副C装备: 卡尔玛青龙刀、虚空杖、法爆。
- 装备优先级: 妖姬三件套 > 努努两件肉装 > 卡尔玛副C装。
- 可接受替代: 妖姬可用巨杀/破防类增伤；卡尔玛可用鬼书/离子补功能。
- 不推荐硬玩装备: 纯物理装、没有羊刀/法爆方向、没有前排肉装时不建议定阵。

## Augments

- 首选海克斯: 1经济 + 1战力 + 1装备的组合。
- 可选海克斯: 维鲁斯优先，其次艾克、阿狸、亚索；复制器和战力强化都可用。
- 不推荐海克斯: 只服务低费赌狗或与法官/重装无关的强化。
- 海克斯选择逻辑: 法官效果和妖姬装备决定上限，经济保证4-1/4-2上8启动。

## Preconditions

- 推荐开局: 法装开，前期佐伊/提莫能带装备，或未来/海魔人连败养经济。
- 装备前置: 羊刀、法爆、大天使方向明确，且能做努努肉装。
- 来牌前置: 4阶段能在8级找到乐芙兰、卡尔玛、努努、俄洛伊体系牌。
- 经济/血量前置: 4-1/4-2能拉8大D；容错不够时不能贪利息。
- 适合转入条件: 法官效果好，妖姬装备成型，重装前排来得顺。

## Avoid

- 不推荐玩的情况: 法官效果不适配、妖姬装备差、4阶段钱少血低、D不到体系四费。
- 同行过多判断: 同行抢妖姬/卡尔玛时，除非你装备和血量领先，否则考虑转木灵飞机或观星物理线。
- 装备冲突: 物理装多应转木灵飞机/观星狙，不要硬玩法系。
- 节奏过晚风险: 8级没全二仍贪上9，容易被成熟阵容入死。

## Tempo

- 前期过渡: 前期不抽卡；连胜可2-1拉4、2-5拉5，连败可用未来战士贪刷新次数。
- 什么时候拉人口: 3-2拉6，4-1或4-2拉8启动。
- 什么时候D牌: 4-1/4-2拉8大D，抽出体系四费全二再停手。
- 什么时候存钱: 妖姬、卡尔玛、努努等关键牌二星后存钱。
- 什么时候转阵: 法装不成或法官效果差时转其他法系/物理线；物理装成型可转木灵飞机。
- 决赛圈调整: 5阶段可上9补2星五费；17.2削弱3星乐芙兰被动伤害，追三星四费仍可但不应作为唯一吃鸡路径。
