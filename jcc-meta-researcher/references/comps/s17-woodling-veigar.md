# S17 木灵小法

- Version: 17.2
- Season: S17 星神
- Tier: 有争议可玩
- Updated: 2026-04-30
- Sources: 代练通S17木灵小法；云手机游戏网S17木灵小法；大地系统/番茄U盘法德比羊阵容；17173转载官方17.2更新公告。
- Confidence: medium-low

## Core

- 核心羁绊: 7木灵族 + 2魔术师 + 2暗星 + 2旅人 + 2堡垒卫士。
- 核心弈子: 小法、库奇、拉莫斯、波比、巴德、卡尔玛、菲兹、小木灵、丽桑卓。
- 成型人口: 8能开核心，9人口补巴德/莫甘娜/高费功能。
- 成型阵容: 小法、库奇、拉莫斯、波比、巴德、卡尔玛、菲兹、小木灵、丽桑卓。
- 可替换弈子: 9人口可挂莫甘娜或高费控制；前排按波比/拉莫斯质量调整。

## Positioning

- 默认站位: 波比/拉莫斯前排，木灵单位分散，小法后排安全位。
- 对刺客/切后排: 小法不要独站角落，巴德/卡尔玛贴近保护。
- 对AOE: 后排错开，避免小法和库奇同时被控。
- 对单点爆发: 主T对位敌方爆发，保护小法启动。
- 站位注意事项: 木灵克隆收益依赖核心活到放技能，别让小法第一波被切。

## Items

- 主C装备: 小法纳什之牙、珠光护手、青龙刀/启动装。
- 主T装备: 波比/拉莫斯日炎、反甲、离子、狂徒、石像鬼。
- 副C装备: 库奇轻语、杀人剑、破防者。
- 装备优先级: 小法启动和爆发装 > 主T肉装 > 库奇物理补伤。
- 可接受替代: 小法可用法爆+科技枪/蓝buff类启动；库奇捡多余物理装。
- 不推荐硬玩装备: 没法系启动、没有前排肉装、小法数量少时不建议强玩。

## Augments

- 首选海克斯: 帮助追三星、经济、装备、木灵相关强化。
- 可选海克斯: 微型妮蔻、宝宝学院、快速三星类强化。
- 不推荐海克斯: 纯后期经济但前期无战力，或完全不帮小法成型的强化。
- 海克斯选择逻辑: 这套更像条件阵容，有小法数量或三星强化时价值更高。

## Preconditions

- 推荐开局: 小法数量多、法装开、木灵牌顺，或拿到明显帮助三星/复制的强化。
- 装备前置: 能做纳什/法爆/启动装，且前排有肉装。
- 来牌前置: 7级或8级前小法数量明显领先。
- 经济/血量前置: 有足够血量在8级搜木灵体系，不适合低血硬追。
- 适合转入条件: 法装太多不适合木灵飞机，同时小法和木灵牌自然来。

## Avoid

- 不推荐玩的情况: 小法少、法装不成、没有三星相关强化、同行也在木灵。
- 同行过多判断: 同行木灵多会影响小法/库奇/拉莫斯质量，优先转飞机或观星。
- 装备冲突: 物理装多应玩木灵飞机，不要强塞小法。
- 节奏过晚风险: 小法迟迟不三，后期会被95阵容压上限。

## Tempo

- 前期过渡: 牧羊/木灵/强二星带法装过渡，不要过早乱D。
- 什么时候拉人口: 常规拉人口到7/8，看小法数量决定是否停留追三。
- 什么时候D牌: 小法数量多时在7或8级追三星；数量少则8级转木灵飞机或观星。
- 什么时候存钱: 小法二星且场面能稳时存钱上8开7木灵。
- 什么时候转阵: 小法不来、物理装多、库奇/锐雯来得多时转木灵飞机。
- 决赛圈调整: 9人口补巴德/莫甘娜/控制，保护小法避免被切。
